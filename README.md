## Configuration

This project now uses a configuration file named `config.json`.

Make sure the file exists in the root directory of the repository with the following content:

{
  "system_title": "App 2.0",
  "status": "CRITICAL"
}

This configuration file is required for the system to run properly.

Note: The previous configuration file `config.txt` has been removed and is no longer used.
